What this is

Xi-Void Tools is a small collection of single-purpose web utilities with a deliberately retro-terminal look. Each tool does one thing, runs entirely in your browser, and asks nothing of you — no sign-in, no tracking, no server.

How it is built

The site is static and built with Jekyll, which GitHub Pages compiles automatically on every push. Shared layout, styling and navigation live in one place (_layouts, _includes and _data), so each page is just its content plus a little front-matter. There is no JavaScript framework and no third-party dependency — every tool is plain, hand-written HTML, CSS and JavaScript.

Design

One stylesheet drives the whole aesthetic. The accent colour is a single CSS variable, which is why the green / amber / cyan switch in the top bar can recolour everything at once. Press / anywhere to jump between tools with the command palette.
