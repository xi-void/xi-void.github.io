What it does

Cipher applies reversible text transforms: ROT13, Caesar shift, Base64, hex and binary. Each one can be run forward or backward.

How to use it

  • Pick a scheme. ROT13 is symmetric; Caesar reveals a shift field; the rest reveal an encode/decode toggle.
  • Type into the input box; the output updates live.
  • swap ↑ moves the output back into the input and flips the direction, so you can round-trip a value.
  • copy output grabs the result.

Notes

These are encodings and obfuscations, not secure cryptography — anyone can reverse them. Base64, hex and binary operate on the UTF-8 bytes of your text. For one-way hashing use Digest; to generate secrets use Keymaker.
