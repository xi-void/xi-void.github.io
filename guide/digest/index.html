What it does

Digest computes a cryptographic hash of your text using the browser's built-in crypto.subtle. A hash is a fixed-length, one-way fingerprint: the same input always yields the same digest, and you cannot reverse it back to the original.

How to use it

  • Choose an algorithm — SHA-256 is the sensible default.
  • Type into the input box; the digest updates live.
  • Switch the output between lowercase and uppercase hex.
  • Press copy digest to grab the result.

Notes

Web Crypto only runs in a secure context, so the tool works over HTTPS or on localhost. Input is encoded as UTF-8 before hashing. SHA-1 is included for compatibility but is no longer collision-resistant — prefer SHA-256 or stronger for anything that matters.
