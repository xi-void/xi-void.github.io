What it does

Notepad is a throwaway buffer for quick text. It saves itself to this browser's localStorage as you type, so your text is still there when you come back — on this device, in this browser.

How to use it

  • Just start typing; a saved note flashes when the buffer is written.
  • download .txt saves the contents to a file.
  • copy all copies everything to the clipboard.
  • clear empties the buffer and removes the saved copy.

Notes

Storage is per-browser and per-device, so the note will not follow you to another machine. Clearing your browser's site data — or using private browsing — removes it. Nothing is ever sent anywhere.
