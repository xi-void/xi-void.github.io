What it does

Keymaker produces random secrets using crypto.getRandomValues, the browser's cryptographically secure random source. It offers two modes: character passwords and word-based passphrases.

How to use it

  • Choose password or passphrase at the top.
  • For passwords, set the length and toggle character sets; enable avoid look-alikes to drop Il1O0o.
  • For passphrases, set the word count, separator, capitalization and whether to append a number.
  • Press generate for a fresh value and copy to grab it.

Notes

The entropy estimate assumes each character or word is chosen independently and uniformly — a useful guide, not a guarantee about any specific output. Generate a new secret for each account and store it in a real password manager. Nothing generated here is transmitted or saved.
