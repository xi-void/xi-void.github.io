What it does

Transmuter takes one line of text and renders it simultaneously in every style it knows — small caps, upside-down, bold, italic, script, monospace, double-struck, circled, full-width and more. Each row is a separate Unicode representation, not a font.

How to use it

  • Type into the plain text box; all styles update live.
  • Press the copy button on any row to copy just that style.
  • Insert example fills in sample text.

Notes

Some styles have gaps in Unicode where a few letters are missing; those fall back to the nearest letterlike symbol or to the plain character. Upside-down text is reversed so it reads correctly when flipped. As always, the final look depends on the font where you paste it.
