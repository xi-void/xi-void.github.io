What it does

Glitch Forge layers Unicode combining diacritical marks onto every character, producing the dripping, corrupted look usually called "zalgo". The marks attach above, through and below each letter.

How to use it

  • Type into the plain text box; output updates as you type.
  • Set corruption from minor to total to control how many marks stack up.
  • Use zones to keep the decay above the line, below it, or both.
  • Press copy output to grab the result.

Notes

Because the effect is built from real combining characters, rendering varies by font and platform, and some apps deliberately limit how many marks they draw. Very heavy corruption can be slow to render in other programs.
