What it does

Countdown is a simple timer. Set a duration and watch it drain; when it reaches zero the display turns red, blinks, and a short tone sounds. Everything runs in the tab — no network, no notifications permission.

How to use it

  • Enter hours, minutes and seconds, or tap a preset (1 / 5 / 10 / 25 min).
  • Press start; use pause to hold and reset to return to the set time.

Notes

The timer tracks real elapsed time, so it stays accurate even if the tab is briefly throttled. The end tone uses the Web Audio API; if your browser blocks audio until you interact with the page, the visual flare still fires. Keep the tab open for the timer to run.
